Description

This test validates that onlevelchange and level attributes exist and are set to correct values, with corresponding events fired, when the charger is plugged in.

Preconditions

  1. The device is unplugged from the charger before this test is run.
  2. The battery must not be full or reach full capacity before the time the test is run.
  1. Plug in the charger and wait for all the tests to complete.
  2. The tests may take long time since the definition of how often the levelchange event is fired is left to the implementation.
