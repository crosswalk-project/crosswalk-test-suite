Preconditions

  1. Plug in USB microphone
  2. Update "sphinx.pulsesrc" section of /etc/speech-recognition/speech-recognition.conf following the output of below command.
    (app user): pactl list sources |grep input |grep "Name:" |grep usb
  3. Reboot test device
  4. Execute following command to init environment:
    (app user) app_launcher -s org.tizen.dialer
    (app user) pacmd set-sink-mute 0 false
  5. Launch test app

TTS Test: Type some words in textbox or default words, then press button "Speak Out".

Test Passes if you can hear the word typed speak out from headset.

STT Test: Press button "Record Voice", speak some words like "play music", then click button "Stop Recording".

Test Passes if words you speak out are recorded and displayed on screen.
