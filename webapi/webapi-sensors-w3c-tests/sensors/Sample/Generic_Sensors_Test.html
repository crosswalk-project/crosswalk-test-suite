Test steps:

  1. Access to https://generic-sensors.appspot.com/
  2. Click the button at the top right corner.
  3. Pop-up shows, and choose the "Ambient light"/"Accelerometer"/"Gyroscope"/"Magnetometer" Sensor type, click "ADD" button
  4. Turn on the switch button
  5. Change the ambient light and rotate the device
    Note:On Mac OS only Ambient Light Sensor is supported

Expected Result:

  1. The "Sensor state" should be "activated"
  2. The "timeStamp" value shows and always grow up.
  3. The "illuminance" value is changed by ambient light.
  4. The X,Y,Z of value is changed by rotate the device.
