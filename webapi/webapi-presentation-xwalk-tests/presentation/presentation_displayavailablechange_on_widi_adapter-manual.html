Widi adapter should be successfully connected with the device, and display mirror screen on the monitor.

  1. Click 'Request Show Presentation' button and then check if

    The 'Display Available' should be 'true';

    The 'Result' should be 'Ok', the 'Received message' should be 'I am from presentation window';

    The secondary display should display: 'Presentation Window showed on Wireless Display' 'Received message: I am from opener window'.

  2. Disconnect Widi adapter (Powser off) and then check if the 'Display Available' should be changed to 'false'.
  3. Reconnect Widi adapter and then check if the 'Display Available' should be changed to 'true'.
  4. If the above steps are all true the test case pass, otherwise it fail.
