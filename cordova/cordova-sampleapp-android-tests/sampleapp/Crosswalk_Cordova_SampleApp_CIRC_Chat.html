Preconditions:

  1. Need a good network

Test steps:

  1. Connect to the IRC server for the first android device
    $ /server irc.freenode.net
  2. Set the nick name in the first android device, e.g. take "test_1" as the nick name
    $ /nick test_1
  3. Connect to the IRC server for the second android device
    $ /server irc.freenode.net
  4. Set the nick name in the second android device, e.g. take "test_2" as the nick name
    $ /nick test_2
  5. Use test_1 to chat with test_2, take "hello" as the chat content
    $ /msg test_2 hello
  6. Check if test_2 receive the "hello" message that test_1 send
  7. Use test_2 to chat with test_1, take "hello" as the chat content
    $ /msg test_1 hello
  8. Check if test_1 receive the "hello" message that test_2 send
  9. test_1 leave the chat
    $ /leave test_2
  10. test_2 leave the chat
    $ /leave test_1
  11. test_1 disconnect to the IRC server
    $ /quit
  12. test_2 disconnect to the IRC server
    $ /quit

Expected Output:

  1. The first android device connect to the IRC server successfully
  2. The first android device set the nick name(e.g. "test_1") successfully
  3. The second android device connect to the IRC server successfully
  4. The second android device set the nick name(e.g. "test_2") successfully
  5. test_2 receive the "hello" message that test_1 send
  6. test_1 receive the "hello" message that test_2 send
  7. test_1 leave the chat successfully
  8. test_2 leave the chat successfully
  9. test_1 disconnect to the IRC server successfully
  10. test_2 disconnect to the IRC server successfully
