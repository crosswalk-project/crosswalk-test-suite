Pre-conditions

  1. Install git
  2. Install gyp by "sudo apt-get install gyp" on Ubuntu or "svn co http://gyp.googlecode.com/svn/trunk build/gyp --revision 1831", add gyp path to PATH
  3. Install crosswalk-app-tools and set crosswalk-app-tools/src to PATH
  4. Add path to aidl to PATH. Example, "/home/belem/adt-bundle-linux/sdk/build-tools/19.0.0"
  5. Install depot_tools by "git clone https://chromium.googlesource.com/chromium/tools/depot_tools.git", add depot_tools path to PATH
  6. cd /path/to/crosswalk binary folder, e.g. /home/belem/crosswalk/canary/crosswalk-9.38.207.0,
    $ chmod +x extension_manager.py

    7. Besides the original environment path config for Android, you may add more PATH contents like:
    /home/belem/crosswalk/canary/crosswalk-9.38.207.0:/home/belem/adt-bundle-linux/sdk/build-tools/19.0.0:/home/belem/github/depot_tools

Test Steps

  1. Enter package source directory, execute terminal command:
    $ ./extension_manager.py --add=https://github.com/crosswalk-project/crosswalk-android-extensions.git
    OR
    $ ./extension_manager.py --add=github:crosswalk-project/crosswalk-android-extensions.git (depends on your github .ssh config)
  2. List all extensions, execute terminal command:
    $ ./extension_manager.py -l
  3. Search extensions, execute terminal command:
    $ ./extension_manager.py --search=crosswalk*
  4. Disable extensions, execute terminal command:
    $ ./extension_manager.py --disable=crosswalk*
    $ ./extension_manager.py -l
  5. Enable extensions, execute terminal command:
    $ ./extension_manager.py --enable=crosswalk*
    $ ./extension_manager.py -l
  6. Usage of crosswalk-pkg, execute terminal command:
    $ crosswalk-pkg --crosswalk=[crosswalk-version] --platforms=android --targets=[arch] [your_project_path]
  7. Remove extensions, execute terminal command:
    $ ./extension_manager.py --remove=crosswalk-android-extensions

Purpose:

Verifies that extension_manager.py should work properly

Expected Result:

  1. The file "<path to extension_manager.py>/extensions/crosswalk-android-extensions/out/Default/gen/iap/iap.jar" exists
  2. Display "+ crosswalk-android-extensions/iap" successfully
  3. Display "+ crosswalk-android-extensions/iap" successfully
  4. Display "- crosswalk-android-extensions/iap" successfully
  5. Display "+ crosswalk-android-extensions/iap" successfully
  6. Generated apk successfully, no error message
  7. Remove the path "<path to extension_manager.py>/extensions/crosswalk-android-extensions" successfully
