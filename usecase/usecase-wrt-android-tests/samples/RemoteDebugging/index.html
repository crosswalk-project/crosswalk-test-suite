Test Steps

  1. Launch the web app named "remotedebugging"
  2. Open Chrome Browser(browser version >= crosswalk version) in host and input "chrome://inspect" in the address bar
  3. Launch the web app named "remotedebugging-no", run the command, then repeat step 2:
    $ adb shell am broadcast -a org.xwalk.intent -e remotedebugging true
  4. Install XWalkCoreShell.apk from the crosswalk-test-apks-<version>-<arch> and launch it, then open any URL
  5. Repeat step 2

Purpose:

Verifies that Crosswalk could debug by remote host when the web app is packaged with remote debugging by default

Expected Result:

  1. Web App could be launched successfully
  2. Web pages or web apps will be shown in the inspection page. Click button 'inspect' to inspect it
  3. The app can not run remote debugging, the result is different from result 2
  4. The expected result is the same as result 2
