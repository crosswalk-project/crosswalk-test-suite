Pre-condition

  • Install the XWalkRuntimeLib.apk (current crosswalk version) on test device

Test Steps

  1. Install the app named like "cookie_shared.apk" and launch it
  2. Type text as Cookie value in "The value of the cookie should be" input box in "Example" section, then click link "Create cookie 1"
  3. Install the app "cookie_ssv.apk" (use "adb install -r *apk" to force upgrading) and launch it
  4. Click link "Read cookie 1" in "Example" section (Please don't click "Erase cookie 1")

Note

  • The app "cookie_ssv" is an upgrade of shared app ("cookie_shared") that change the app version from lower version to higher version for testing cookie
  • The app "cookie_shared" is the lower version shared app
