Test Steps

  1. Using an absolute path to run the steps:
    1. Execute terminal command in the directory where make_apk.py:
      $ python <absolute path of make_apk.py>/make_apk.py --name=[name] --package=[package] --app-url=http://www.baidu.com --arch=x86
    2. Install the packaged apk
    3. Launch the web app
    4. Change work directory, execute terminal command in another directory:
      $ python <absolute path of make_apk.py>/make_apk.py --package=[package] --manifest=[manifest-path] --arch=arm
    5. Install the packaged apk
    6. Launch the web app
    7. Change work directory, execute terminal command in another directory:
      $ python <absolute path of make_apk.py>/make_apk.py --package=[package] --app-local-path=index.html --app-root=[app-root-path]
    8. Install the packaged apk
    9. Launch the web app
  2. Set the PATH variable and run the steps from a different directory:
    1. Execute terminal command in the directory where the file make_apk.py:
      $ chmod +x make_apk.py
      $ export PATH=$PATH:<absolute path of make_apk.py>
      $ make_apk.py --name=[name] --package=[package] --app-url=http://www.baidu.com --arch=x86
    2. Install the packaged apk
    3. Launch the web app
    4. Change work directory, execute terminal command in another directory:
      $ make_apk.py --package=[package] --manifest=[manifest-path] --arch=arm
    5. Install the packaged apk
    6. Launch the web app
    7. Change work directory, execute terminal command in another directory:
      $ make_apk.py --package=[package] --app-local-path=index.html --app-root=[app-root-path]
    8. Install the packaged apk
    9. Launch the web app

Purpose:

Verifies that make_apk.py should be runnable from any location

Expected Result:

  1. Step 1
    1. Apk is generated successfully
    2. Web app is installed successfully
    3. Web app is launched successfully
    4. Apk is generated successfully
    5. Web app is installed successfully
    6. Web app is launched successfully
    7. Apk is generated successfully
    8. Web app is installed successfully
    9. Web app is launched successfully
  2. The results same as step 1
