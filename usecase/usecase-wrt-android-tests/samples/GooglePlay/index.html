Test that Crosswalk shared mode app can get matched Crosswalk runtime library from Google Play store.

Pre-condition:

  1. Uninstall pre-installed Crosswalk runtime library.
  2. Google Play store or other PRC store are installed.

Test steps:

Device ArchitectureApp ModeExpected Crosswalk installed via Google Play StoreExpected Crosswalk installed via PRC Store
ARM 32bitSharedorg.xwalk.coreorg.xwalk.core
ARM 64bitSharedorg.xwalk.core64org.xwalk.core64
IA 32bitSharedorg.xwalk.coreorg.xwalk.core.ia
IA 64bitSharedorg.xwalk.core64org.xwalk.core64.ia
  1. Find(or Build) a Crosswalk shared mode apk, install it to test devices following the table, and launch it.
  2. Get Crosswalk via Google Play store or other PRC store.
  3. Launch the app again.
  4. Check installed Crosswalk runtime library by command line.
  5. Uninstall the installed Crosswalk runtime library.

Reference command lines:

  1. Building shared mode apk:
    $ crosswalk-pkg --crosswalk=[crosswalk-version] --platform=android --android=shared --targets=arm /path/to/testapps/demo
  2. Finding installed Crosswalk runtime library:
    $ adb shell dumpsys|grep org.xwalk.core
