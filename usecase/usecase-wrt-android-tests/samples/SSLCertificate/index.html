Pre-condition:

  1. Connect to available network.
  2. Install mitmproxy in host device by command line: $ pip install mitmproxy
    More offical information: https://mitmproxy.org/

Test steps:

  1. Install sslcertificate.apk, and lunch it.
  2. Click the baidu.com and anr.io
  3. Start mitmproxy on host device:
    $ mitmproxy -e -U http://<proxy-host>:<proxy-port>
  4. Connect test device to the same network with host device, and set the proxy to host ip address, proxy port is 8080.
    PS: If use Android emulator for testing, start it with: emulator -avd <name> -http-proxy 127.0.0.1:8080
  5. Restart the sslcertificate app, click the baidu.com and anr.io again.
