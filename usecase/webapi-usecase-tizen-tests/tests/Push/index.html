Check if PushClient gets messages from the push server

A network connection capable of accessing the Internet MUST be established


1. Install and launch PushClient(TestPush) Application

(Make sure the device is online and the applications shows registration success popup.)

2. Press Hide in the PushClient

3. Back to this page and fill in a push message and click Push button

4. Check PushClient becomes foreground showing PASS text

5. Exit PushClient

6. Back to this page and fill in another message and click Push button

7. Check a notification pops up and PushClient launchs with PASS text when the notification is clicked
