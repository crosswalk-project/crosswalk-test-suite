Test Purpose:

Verifies the interfaces and methods of calendar is supported and valid.

Test Steps:

  1. when open the page,if the page show 'undefined' means the tizen.calendar is undefined and you can not do the following steps.
  2. Click the 'Add' button, check if the calendar can add an item synchronously.
  3. Click the 'addBatch' button, check if the calendar can add an array of items asynchronously.
  4. Click the 'Update' button, check if an existing item in the calendar can be updated.
  5. Click the 'updateBatch' button, check if an array of existing items in the calendar can be updated.
  6. Click the 'Remove' button, check if an item can be removed from the calendar that corresponds to the specified identifier.
  7. Click the 'removeBatch' button, check if several items can be removed from the calendar asynchronously depending on the specified identifiers
  8. Click the 'Find' button, check if an array of CalendarItem objects for items stored in the calendar can be found.
  9. Click the 'Clone' button, check if the CalendarItem object can be cloned, detached from any calendar.

Expected Result:

Test passes if interfaces and methods of calendar is supported and valid normally.
