    Check if data is successfully synchronized with an OMA DS server


    Precondition: Pick one in <Sample OMA DS Server List> and add test contacts

    <Sample OMA DS Server List>

    - http://my.funambol.com/sync

    - https://www.everdroid.com/web

    - OMA DS 1.2 protocol Server


    1. Fill in preset OMA DS Server, ID, Password and Database URI containing data to sync.
    For the Database URI field "OMA DS Server Database URI",
    - if you choose funambol OMA DS Server for this test, a tester needs to fill "card" string to the URI field.
    - if you choose everdroid OMA DS Server for this test, a tester needs to fill "con" string to the URI field.
    - if you are using other OMA DS Servers, then you need to go to the web page of the OMA DS Server which you use for the test.

    2. Click Add Profile

    3. Click Start Sync

    4. Check the contacts are synchronized with the OMA DS server
