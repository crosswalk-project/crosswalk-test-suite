Test Purpose:

Verifies the functionality of launching the Application which control by user instead of system. Tests the two test Applications could be launched by App ID or customized AppControl's "operation" attribute by user.

Test Step:

  1. Click the "Install" button to install the custom package.
  2. Click the "Launch" button to launch the custom package.
  3. Click the "Install" button to install the regular package.
  4. Click the "Launch" button to launch the regular package.
  5. Click the "Uninstall" button to uninstall the two widgets.

Expected Result:

Test passes if the test Applications launch successfully.
