Preconditions:

Debian testing is the only tested platform. Node.js, the debuild tool, the Crosswalk runtime, and git must be functional

  1. Download Crosswalk app tools: git clone https://github.com/crosswalk-project/crosswalk-app-tools.git
  2. Checkout the iOS backend: cd node_modules, then git clone https://github.com/crosswalk-project/crosswalk-app-tools-deb.git crosswalk-app-tools-backend-deb
  3. Install dependencies: cd crosswalk-app-tools-backend-deb, then npm install, and cd ../..
  4. The main script is crosswalk-app-tools/bin/crosswalk-app. Set environment PATH or invoke with directory

Test Steps:

  1. Execute command "crosswalk-app create com.example.packingtool"
  2. Copy web application resources from "res/testapp/org.packagemanage.packingtool"to "com.example.packingtool/app", enter directory "com.example.packingtool", then execute command "crosswalk-app build"
  3. Execute command "dpkg -i packingtool***.deb" to install web application
  4. Execute command "packingtool" to launch webapp
  5. Execute command "dpkg -P packingtool" to uninstall webapp

Expected Result:

  1. Execute command successfully, directory named "com.example.packingtool" should be created
  2. Execute command successfully, webapp should be generated in "com.example.packingtool/pkg"
  3. Webapp should be installed successfully
  4. Webapp should be launched successfully
  5. Webapp should be uninstalled successfully

Verifies tool support for packing crosswalk web applications into deb package
