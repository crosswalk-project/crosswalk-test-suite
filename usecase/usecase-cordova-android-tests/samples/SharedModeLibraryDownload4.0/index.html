Precondition

  1. Put XWalkRuntimeLib.apk in a server and make sure the test device can access it using [http://host/XWalkRuntimeLib.apk]
  2. Make sure the test device has not install XWalkRuntimeLib.apk
  3. Require npm
  4. Require the latest Cordova CLI, and must >= 5.0.0
    $ sudo npm install cordova -g
  5. Connect the android device to the host

Test Steps

  1. Enter 'usecase-cordova-android-tests/res/SharedModeLibraryDownload' directory
  2. Run the python script:
    $ python test.py -u [http://host/XWalkRuntimeLib.apk]
  3. Check the command output
  4. Check the app on the device and click "Get Crosswalk" button, check the screen
  5. Click "Install", then check the screen
  6. Click "Done", then check the screen
  7. Quick press back button to exit 'CordovaPackage' app after the app launch
  8. Uninstall the app

Expected Results:

  1. The command run successfully, no error
  2. The console display 'Build Package Successfully' and 'Package Name Consistent'
  3. The app is running and display message "Please get Crosswalk Runtime Library"
  4. Start to download the XWalkRuntimeLib.apk
  5. Display "App installed"
  6. The app can run successfully
  7. The app can exit successfully
  8. The app can be uninstalled successfully
