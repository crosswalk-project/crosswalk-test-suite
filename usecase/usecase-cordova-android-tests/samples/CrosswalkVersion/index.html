Precondition

  1. Require the latest Cordova CLI, and must >= 5.0.0
    $ sudo npm install cordova -g
  2. Install BeautifulSoup
    $ sudo pip install BeautifulSoup

Test Steps

  1. Enter 'usecase-cordova-android-tests/res/CrosswalkVersion' directory
  2. Run the python script:(take 'arm' as example, if your device is x86, need to use 'x86' instead of 'arm')
    For embedded mode: $ python test.py -m embedded -a arm
    For shared mode: $ python test.py -m shared -a arm
    For lite mode: $ python test.py -m lite -a arm
  3. Check the command output
  4. Check if there are several apk files in 'usecase-cordova-android-tests/res/
    CrosswalkVersion directory
    For stable channel, there should be 14 apk file, and the apk names start with 'CrosswalkVersion_stable'
    For canary channel, there should be 6 apk file, and the apk names start with 'CrosswalkVersion_canary'
    For beta channel, there should be 4 apk file, and the apk names start with 'CrosswalkVersion_beta'
  5. Install and launch these apks one by one
  6. Check if 'Expected Crosswalk Version' is the same with 'Actual Crosswalk Version' on each app homepage
