Precondition

  1. Put XWalkRuntimeLib.apk in a server and make sure the test device can access it using [http://host/XWalkRuntimeLib.apk]
  2. Make sure the test device has not install XWalkRuntimeLib.apk
  3. Connect the android device to the host

Test Steps

  1. Enter 'usecase-cordova-android-tests/res/SharedModeLibraryDownload' directory, copy test.py files to crosswalk-cordova directory
  2. Run the python script:
    $ python test.py -u [http://host/XWalkRuntimeLib.apk]
  3. Check the command output
  4. Check the app on the device and click "Get Crosswalk" button, check the screen
  5. Click "Install", then check the screen
  6. Click "Done", then check the screen
  7. Quick press back button to exit 'CordovaPackage' app after the app launch
  8. Uninstall the app
