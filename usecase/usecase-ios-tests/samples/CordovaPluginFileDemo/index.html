Precondition

  1. Connect iOS devices to your localhost

Test Steps

  1. Run below commands to build project under crosswalk-ios directory:
    $ cd Demos/Cordova
    $ open CordovaPluginFileDemo.xcworkspace
  2. Select the project "CordovaPluginFileDemo" --> set the active scheme --> CordovaPluginFileDemo --> <iOS Device> -- > Product --> Build
  3. Check the device and launch the app named like "CordovaPluginFileDemo"
  4. Write some characters to the input, then click "Write"
  5. Click "Read"
  6. Click "Delete"
