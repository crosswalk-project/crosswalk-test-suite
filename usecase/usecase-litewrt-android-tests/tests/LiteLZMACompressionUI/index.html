Test Steps

  1. Pack an apk with Crosswalk Lite binary
  2. Install the packaged apk on device
  3. Launch the app on device

Purpose:

  • Verifies when pack an apk and run the app, the compression and decompression work properly or not, at the same time the progress bar UI display normal or not

      • Expected Result:

      1. When pack the apk, the LZMA compression works fine
      2. Install the packaged apk successfully
      3. Launch the app first time and found pop up a progress bar with message('Runtime is preparing..'), the progress bar has no truncation or other display issue
      4. When decompression, the decompression time is reasonable and the progress bar will disappear
      5. App runs successfully
