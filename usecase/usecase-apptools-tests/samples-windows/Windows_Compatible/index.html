Test Steps

  1. If org.test.foo exists in usecase-apptools-tests/tools/ directory, remove it
    Then run below commands under usecase-apptools-tests/tools/ directory:
    On Windows 10:
        New "org.test.foo" folder
        $ cd org.test.foo
        $ node ..\crosswalk-app-tools\src\crosswalk-pkg --platforms=windows --manifest="org.test.foo" --crosswalk="path to xwalk win zip" .\
  2. Install foo app on Windows 8.1
  3. Find foo app on windows 8.1
  4. Launch the app on Windows 8.1
  5. Check the homepage of foo app
  6. Uninstall the app when it is running
  7. Install and launch the foo app again. Close foo app via clicking the close button, and kill foo app from Task Manager
  8. Uninstall foo app on windows 8.1

Purpose:

Verifies Windows 10's app could be launched on Windows 8.1

Expected Result:

  1. Build org.test.foo successfully
  2. Install the foo app successfully
  3. Find the foo app successfully
  4. Launch the foo app successfully
  5. You can see "This is a Crosswalk Sample Application" on the device
  6. Unistall the foo app successfully when the app running
  7. Stop the foo app successfully
  8. Unistall the foo app successfully
