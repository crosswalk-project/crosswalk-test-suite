Precondition

  1. On Linux or OS X host:
        Remove environment PATH with 'crosswalk-app-tools/src/crosswalk-pkg'.
        Install crosswalk-app-tools by run command:
        $ sudo npm install -g crosswalk-app-tools
    On Windows host:
        Remove crosswalk-pkg environment.
        Install crosswalk-app-tools by run command:
        $ npm install -g crosswalk-app-tools

Test Steps

  1. If org.test.foo exists in usecase-apptools-tests/tools/ directory, remove it
    Then run below commands under usecase-apptools-tests/tools/ directory:
    New org.test.foo folder
    $ cd org.test.foo
    Copy ../../testapp/start_url/index.html to current directory
    Shared Mode:
        $ crosswalk-pkg --manifest="org.test.foo" --android=shared --targets="armeabi-v7a/arm64-v8a x86/x86_64" ./
    Embedded Mode:
        $ crosswalk-pkg --manifest="org.test.foo" --targets="armeabi-v7a/arm64-v8a x86/x86_64" ./
    Lite Mode:
        $ crosswalk-pkg --manifest="org.test.foo" --android=lite --targets="armeabi-v7a/arm64-v8a x86/x86_64" ./
  2. Shared Mode:
        Install shared mode foo app on your android device which is "arm" architecture
        Install shared mode foo app on your android device which is "x86" architecture
    Embedded Mode or Lite Mode:
        Install arm foo app on your android device which is "arm" architecture
        Install x86 foo app on your android device which is "x86" architecture
  3. Check icon of foo app in your android device
  4. Launch foo app in your android device
  5. Check the homepage of foo app
  6. Uninstall the app when it is running
  7. Install and launch the foo app again. Close foo app via clicking the home button, and kill foo app from Task Manager
  8. Long press the icon of foo app to uninstall it
  9. On Linux or OS X host:
        Uninstall crosswalk-app-tools by run command: $ sudo npm uninstall -g crosswalk-app-tools
        Set environment PATH or invoke with "crosswalk-app-tools/src/crosswalk-pkg" directory
    On Windows host:
        Uninstall crosswalk-app-tools by run command: $ npm uninstall -g crosswalk-app-tools
        Create 'crosswalk-pkg' env, invoke with 'crosswalk-app-tools/src/crosswalk-pkg' directory

Purpose:

Verifies foo app can run successfully

Expected Result:

  1. Build org.test.foo successfully
  2. Install the foo app successfully
  3. Find the foo app successfully
  4. Launch the foo app successfully
  5. You can see "This is a Crosswalk Sample Application" on the device
  6. Unistall the foo app successfully when the app running
  7. Stop the foo app successfully
  8. Unistall the foo app successfully
  9. Uninstall AppTools and set the env successfully
