Precondition

  1. The Node.js, the debuild tool, the Crosswalk runtime, and git must be functional
  2. Download crosswalk-app-tools and configure it
    • cd tools, then download Crosswalk app tools: git clone https://github.com/crosswalk-project/crosswalk-app-tools.git
    • Initialize the Crosswalk app tools: cd crosswalk-app-tools , then npm install
    • Checkout the deb backend: cd node_modules , then git clone https://github.com/crosswalk-project/crosswalk-app-tools-deb.git crosswalk-app-tools-backend-deb
    • Install dependencies: cd crosswalk-app-tools-backend-deb , then npm install , and cd ../..

Test Steps

  1. If org.test.foo exists in usecase-apptools-tests/tools/ directory, remove it
  2. Then run below commands under usecase-apptools-tests/tools/ directory:
    • $ ./crosswalk-app-tools/src/crosswalk-app create org.test.foo
    • $ cd org.test.foo
    • $ ../crosswalk-app-tools/src/crosswalk-app build
    • Install /path/to/org.test.foo/pkg/foo_0.0.1-1_amd64.deb on depin
    • Find foo app in depin
    • Launch foo app in depin
    • Check the homepage of foo app
    • Close foo app via clicking the close button
    • Uninstall foo app

Purpose:

Verifies foo app can run successfully

Expected Result:

  1. Install the foo app successfully
  2. Find the foo app successfully
  3. Launch the foo app successfully
  4. You can see "This is a Crosswalk Sample Application" on the page
  5. Stop the foo app successfully
  6. Unistall the foo app successfully
