Test Steps

  1. Pack the web app with several options, command like:
    $ python make_apk.py --name=[name] --package=[package] --app-root=app_path --app-local-path=index.html --app-version=1.0.0 --description=YourApplicationDescription --fullscreen --mode=embedded --orientation=landscape --target-dir=/home/path/to/
  2. Check the web app version
  3. Check the packer tool folder
  4. Check the value of "android:description" in /app/path/AndroidManifest.xml
  5. Check the description in /app/path/res/values/string.xml
  6. Install the packaged apk
  7. Launch the web app

Purpose:

Verifies that Packaging Tool support the 'app-version', 'app-local-path', 'name', 'package', 'description', 'fullscreen', default 'icon', 'mode=embedded', 'orientation=landscape', 'target-dir' and default 'arch' options

Expected Result:

  1. Two apks is created in the target dir successfully
  2. The version of the web app is 1.0.0
  3. An app folder is generated successfully in packer tool
  4. Variable path for description can be found
  5. Description of the web app is "YourApplicationDescription"
  6. Web App could be installed successfully
  7. Web App could be launched with fullscreen and screen orientation is landscape
