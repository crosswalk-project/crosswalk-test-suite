Image Id:
Open Image

Purpose:

Verifies that enable scheme content

Pre-condition:

  1. There is at least 1 image in the device
  2. Connect a device to host by usb

Test Steps:

  1. Run command in terminal:
    $ adb shell content query --uri content://media/external/images/media"
  2. Set the "_id" value to "Image Id" textarea
  3. Click the link "Open Image"

Expected Result:

  1. List all image id successfully
  2. Input "_id" value to "Image Id" textarea successfully
  3. Selected image is displayed successfully
