Pre-condition:

  • Connect to network
  • The plugin inappbrowser and network-information have been installed

Purpose:

Verify the inappbrowser can work well

Test Steps:

  1. Click the button "Open remote PDF(target=Default)"
  2. Click the button "Open local PDF(target=Default)"
  3. Click the button "Open remote PDF(target=_self)"
  4. Click the button "Open local PDF(target=_self)"

Expected Results:

  1. Open a document list
  2. Open local PDF
  3. Open a document list
  4. Open local PDF
