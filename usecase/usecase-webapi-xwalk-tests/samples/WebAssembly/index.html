Testing on Android:

  1. Launch sub app "Web Assembly Demo", choose "Play WebAssembly" and wait for loading successfully.
  2. Connect an external gamepad to the device.
  3. Verify if the demo UI can full load and show high-definition.
  4. Verify if the robot can move smoothly.

Testing on Windows:

  1. Launch sub app "Web Assembly Demo", choose "Play WebAssembly" and wait for loading successfully.
  2. Verify if the demo UI can full load and show high-definition.
  3. Verify if the robot can move smoothly by W, A, S, D or the arrow keys.
  4. Verify if the robot can fire by the mouse.

Testing on Iot:

  1. Run command in terminal:
    $ xwalk --no-sandbox https://webassembly.github.io/demo/ -js-flags=--expose_wasm
  2. Choose "Play WebAssembly" and wait for loading successfully.
  3. Verify if the demo UI can full load and show high-definition.
  4. Verify if the robot can move smoothly by W, A, S, D or the arrow keys.
  5. Verify if the robot can fire by the mouse.
