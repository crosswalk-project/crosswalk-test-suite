Test Steps:

  1. Create /opt/share folder on device
  2. Install the webapp setting-encrypt-enable.wgt and setting-encrypt-disable.wgt in testapp folder.
  3. "ls /home/app/.config/xwalk-service/applications/'enable_pkgid'" to list the webapp file
  4. "cat /home/app/.config/xwalk-service/applications/'disable_pkgid'/index.html" to show the webapp index.html
  5. Uninstall the widget.

Purpose:

Confirm the widget application is encrypted or not.

Expected Result:

  1. Folder can be created successfully.
  2. The webapp can be installed successfully.
  3. The webapp file can not be listed successfully.
  4. The index.html is showing normally.
  5. Webapp can be uninstalled successfully.
