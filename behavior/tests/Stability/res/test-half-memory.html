Pre-condition:

Make sure the free storage is more than half memory.

Test Step:

  1. Change the "test-half-memory.wgt".
    1. Open the folder of "test-half-memory"(suppose you have the "test-half-memory.wgt" or you can find it in the behavior rpm, unzip the wgt as "test-half-memory").
    2. Push a file that the size is larger than half memory into the folder and rename the file as "zipzip.png".
    3. Use the command :"zip -rq ../test-half-memory.wgt *" to create a "test-half-memory.wgt".
    4. Use the command: "app_launcher -l" in "app" user to get the app id of tct-behavior-tests.
    5. Push the created widget to device with the command "sdb push 'path'/test-half-memory.wgt /home/TIZEN_USER/apps_rw/xwalk/applications/'app-id'/tests/Stability/res/".
  2. Click the "Install" button to install the widget.
  3. Click the "Uninstall" button to uninstall the widget.

Expected Result:

Test passes if the installation is successful.
