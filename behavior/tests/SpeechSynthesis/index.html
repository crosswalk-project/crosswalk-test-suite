Test Purpose:

Verifies SpeechSynthesis realated interface work well.

Pre-condition:

  1. Requires the network connection that could connect to speech synthesis service.

Test steps:

  1. Press "start" button to speak the default text, you can input text if you want to.
  2. Press "cancel" button to stop the sound.
  3. Press "pause" button to pause the sound.
  4. Press "resume" button to resume the sound.

Expected Result:

  1. Can hear a sound and its content must match with the text by "start" button.
  2. Can stop the sound by "cancel" button.
  3. Can pause the sound by "pause" button.
  4. Can resume the sound by "resume" button.
