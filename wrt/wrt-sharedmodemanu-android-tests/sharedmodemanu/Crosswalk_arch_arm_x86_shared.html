Precondition:

  1. No Crosswalk Runtime Library installed in devices

Test steps:

  1. Open one terminal command line window run terminal command, just need to run one of the commands below:
    • $ python make_apk.py --name=<name> --package=<package> --app-url=http://www.baidu.com --mode=shared
    • $ python make_apk.py --package=<package> --manifest=<manifest-path> --mode=shared
    • $ python make_apk.py --package=<package> --app-local-path=index.html --app-root=<app-root-path> --mode=shared
  2. Install the x86 apk on x86 devices
  3. Launch web app
  4. Click "Get Crosswalk" to download the Crosswalk Library and install it
  5. Check the launched screen
  6. Repeat the steps 2, 3, 4, 5 for the arm apk on arm devices

Expected Output:

  1. Generate APK installation package successfully
  2. Install web app successfully
  3. Launch web app failed, popup a message like "Please get Crosswalk Runtime Library"
  4. Install the Crosswalk Library successfully
  5. Launch the app successfully
  6. The results same with results 2,3,4,5
