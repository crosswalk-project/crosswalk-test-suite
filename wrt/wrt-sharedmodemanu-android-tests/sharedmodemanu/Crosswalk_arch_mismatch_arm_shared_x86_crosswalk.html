Precondition:

  • Install RuntimeLib for x86 on arm devices

Test steps:

  1. Open terminal command window to run the command:
    • $ crosswalk-pkg --crosswalk=[crosswalk-version] --platforms=android --android=shared --targets=arm [app-root-path]
  2. Install apk on arm devices
  3. Launch web app
  4. Click "Get Crosswalk" to download the Crosswalk Library and install it
  5. Check the launched screen

Expected Output:

  1. Generate APK installation package successfully
  2. Install web app successfully
  3. Launch web app failed and prompt: Please get Crosswalk Runtime Library
  4. Install the Crosswalk Library successfully
  5. Launch the app successfully
