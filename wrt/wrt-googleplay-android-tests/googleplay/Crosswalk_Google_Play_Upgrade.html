Pre-condition:

  1. A test device with x86 architecture and make sure it has Goole Play installed, and be able to access Google Play
  2. An available google play developer account

Test steps:

  1. Sign in the Google Play Developer Console on browser and click the button "Add new application"
  2. Input a title as application name and click the button "Upload APK"
  3. Choose "ALPHA TESTING" or "BETA TESTING" item and click the button "Upload new APK to Beta", then upload the "test_upgrade.apk" into the application
  4. Click the link "Manage list of testers", then add Google Groups or Google+ Communities info
  5. Fill in the required information in "Store Listing" and "Pricing & Distribution", then publish the application(Note: it will take some time to audit the application from google play)
  6. Click the link "Manage list of testers" and copy the link which should be shared with testers, then forward to the link to become a tester
  7. Click the link "View in Google Play store", then click the button "Add to Wishlist"
  8. Login the "google play store" in the test device and choose Wishlist
  9. Install the application which you published
  10. Launch the application which you published
  11. Check the application version from "Application Manager" in the test device
  12. Upgrade the application which you published with "test_upgrade/test_upgrade.apk", then publish the application(Note: it will take some time to audit the application from google play)
  13. Login the "google play store" in the test device and choose the application you published in "My apps"
  14. Click "UPDATE" button
  15. Launch the application which you published
  16. Check the application version from "Application Manager" in the test device

Expected Output:

  1. A dialog: "Add new application" is popuped
  2. The application is created successfully with the name which you inputed
  3. The apk is uploaded into the application successfully
  4. Add Google Groups or Google+ Communities successfully
  5. The application is published successfully
  6. Become a tester for the application which you published successfully
  7. The application which you published is added to your Wishlist successfully
  8. Login the "google play store" and the wishlist is displayed successfully
  9. The application which you published is installed successfully
  10. The application which you published is launched successfully
  11. The application version is 1.0.0
  12. The application is upgraded successfully
  13. The application is showed in "My apps"->"Updates" section
  14. The application is updated successfully
  15. The application launched successfully
  16. The application version is 1.0.1
