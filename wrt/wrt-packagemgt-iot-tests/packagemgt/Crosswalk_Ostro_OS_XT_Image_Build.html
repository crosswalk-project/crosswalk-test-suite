Precondition:

Hardware

  1. RAM memory: >= 8GB
  2. Disk: Make sure more than 50GB free space.

Software

  1. Ubuntu Linux 14.04+, x86_64
  2. GCC: 4.9+
  3. The packages gawk chrpath texinfo are installed: 
    sudo apt-get install gawk chrpath texinfo
    4. 
    
  4. (Optionally) Docker installed
    5. 
    
  5. (Optionally) Add the current user to docker group:
    sudo gpasswd -a ${USER} docker
    6. 
  

  

    Test steps:
  

  
    
    
    
      
  1. Build Ostro OS XT image with Crosswalk:
    
        
    $ ./ostro-os-xt-xwalk-build.sh
    
        Optionally, if you want to build the image with docker, comment the last line of ostro-os-xt-xwalk-build.sh:
        
     # bitbake ostro-xt-image-noswupd
    
        Add a new line:
        
    docker/local-build.sh
    
        Then run the ostro-os-xt-xwalk-build.sh script.
      
    2. 
    
    
    
    
      
  2. Check if the image works well
    3. 
        
      
          
    1. Flash the image to a Joule device
      2. 
          
    2. Launch a Crosswalk hello world application on Joule device
      $ xwalk <path/to/manifest>
      3. 
        
    
    
    
  

  

    Expected Results
  

  
    
    
  1. No error during the build.
    2. 
    
  2. Ostro OS XT image generated successfully at 
    ostro-os-xt/build/tmp-glibc/deploy/images/<CPU arch>
    3. 
    
  3. The Joule can boot normally.
    4. 
    
  4. The Crosswalk hello world application can launch normally.
    5.
