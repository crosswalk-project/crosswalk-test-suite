Test steps:

  1. Make sure there is at least 1 image in the device.
  2. Connect the device to Ubuntu.
  3. Run command in terminal: "adb shell content query --uri content://media/external/images/media"
  4. Check the terminal and get a value of "_id".
  5. Set the "_id" value to "Image Id" textarea.
  6. Click the link of "Open Image".

Test passes if the image display when click the link of "Open Image".
