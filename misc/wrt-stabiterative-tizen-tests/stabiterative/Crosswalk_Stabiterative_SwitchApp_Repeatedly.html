Test steps:

  1. Install webapp by typing "pkgcmd -i -t xpk -q -p lifecycle-nonpers-multiwin-tests.xpk",you can get xpk from wrt-common-webapp-tests suite.
  2. Click app icon to launcher web app.
  3. Press Home key let web app in the background.
  4. Click native app icon to launch it ,such as Message.
  5. Long press Home key into background.
  6. Click first web app to switch to the front desk.
  7. Switch between native app and web app repeatedly.

Expected Output:

  1. Webapp is installed successfully, no error, no warning message.
  2. The app is launched successfully.
  3. The webapp into the background.
  4. Then native app launched successfully
  5. Access background via long press home key
  6. The webapp should be switch to front desk by click webapp in background list
  7. No error when operation repeatedly
