Image Id:
Open Image

Pre_condition:

  1. Make sure there is at least 1 image in the device.
  2. Connect the device to Ubuntu.
Test steps:

  1. Run command in terminal: "adb shell content query --uri content://media/external/images/media"
  2. Check the terminal and get a value of "_id".
  3. Set the "_id" value to "Image Id" textarea.
  4. Click the link of "Open Image".

Expected Output:

  1. List all image id
  2. Check the image id you want get ok
  3. Put one image id to textarea ok
  4. The image display when click the link of "Open Image"
