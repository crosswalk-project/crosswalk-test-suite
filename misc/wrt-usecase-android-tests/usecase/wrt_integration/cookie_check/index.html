cookietester

Make sure device is connected to network

Test steps:

  1. Click cookietester link and input new Cookie name & value
  2. Click "Set Test Cookie"
  3. Continue to add new cookies for 3 times
  4. Select one exist cookies name, copy to "Cookie name" input box, and changes the value its value in "Cookie value" input box, then click "Set Test Cookie" to update it
  5. Click "Remove Cookie" button

Expected Output:

  1. Cookie name and value can be input
  2. New cookie is added and showing correctly
  3. All cookies are adding and showing correctly
  4. After Set Test Cookie the Cookies value can be update successfully
  5. Cookies can be removed successfully
