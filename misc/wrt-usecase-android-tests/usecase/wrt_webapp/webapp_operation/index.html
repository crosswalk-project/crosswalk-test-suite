SINA

Pre_condition:

Make sure device is connected to network

Test steps:

  1. Click SINA link to open new window
  2. Check basic function,such as button click and drag etc
  3. Press Back key back to main window.
  4. Press Back key back to close webapp.
  5. Uninstall the webapp

Expected Output:

  1. One new window opened successfully
  2. Basic function work fine and interface display normally
  3. Can be back to main window successfully
  4. Web app can be close successfully
  5. Web app is able to uninstall successfully
