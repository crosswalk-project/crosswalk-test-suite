Pre-condition

  1. Install WPS or other thirdpart app to open docx/png/pdf file
  2. Make sure there is a sdcard folder named "/mnt/sdcard" or "/mnt/sdcard1" in test device, just need one of them

Test Steps

  1. Launch web app named "openfile" in your device
  2. Click link "DOCX file" to open doc file
  3. Click link "PNG file" to open png file
  4. Click link "PDF file" to open pdf file

Purpose

Verifies that enable open a file with third part app

Expected Results

  1. The web app is launched successfully
  2. DOCX file can be opened successfully
  3. PNG file can be opened successfully
  4. PDF file can be opened successfully
