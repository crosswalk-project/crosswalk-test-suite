Scan for media renderers

Test Purpose:

Check if can scan for media renderers and operate media renderers successfully

Precondition:

Make sure there is a media server in network

Test Steps:

  1. Click the 'Scan for media renderers', waiting for scaning media renderers successfully
  2. Perform following operations for scanned media contents: 'Open', 'Prefetch', 'Play', 'Pause', 'Next', 'Previous', 'Mute', 'Set Speed', 'Set Volume', 'Goto Track', 'Cancel', 'Stop'

Expected Result:

Test passes if scan for media renderers and operate media renderers successfully
