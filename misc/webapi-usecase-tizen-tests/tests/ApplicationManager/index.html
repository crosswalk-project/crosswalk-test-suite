Test Purpose:

Verifies the Applicationinformation and ApplicationContext interface are valid.

Verifies ApplicationManager.launch/kill/getCurrentApplication/getAppContext/getAppsContext methods are supported and valid.

Verifies ApplicationMnager.addAppinfoEventListener and install/uninstall/update events are supported and valid.

Test Steps:

  1. Run the application or click 'AllApp' button, check if the application list can be shown
  2. Click the 'AppInfo' link, check if the application information which you select can be shown
  3. Click the 'Run App' link, check if the application which you select can be run and success infromation can be shown
  4. Click the 'CurrentApp' button, check if current application information can be shown
  5. Click the 'AllContext' button, check if contextId and appId of running applications can be shown
  6. Click the 'Kill App' link, check if specified applicaion can be closed
  7. Click the 'CurrentContext' button, check if contextId and appId of current application can be shown

Expected Result:

Test passes if application information, application context and event notification can be got, kill and launch application normally.
