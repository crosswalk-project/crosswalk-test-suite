Test that 32bit Crosswalk app with shared mode is installed in 64-bit device, the device will consider the app is for first primary instruction set what the device supports. For the 64-bit device, it still determines the app is for 64-bit.

Test steps:

Note: Crosswalk(Vn) is Crosswalk stable version.
NumberDemo Crosswalk VersionCrosswalk Runtime Lib Version
132bit Crosswalk(Vn)64bit Crosswalk(Vn)
232bit Crosswalk(Vn)64bit Crosswalk(Vn-1)
332bit Crosswalk(Vn)64bit Crosswalk(Vn-2)
432bit Crosswalk(Vn-1)64bit Crosswalk(Vn)
532bit Crosswalk(Vn-2)64bit Crosswalk(Vn)
  1. Build "demo" app with 32-bit Crosswalk library following the table:
    $ crosswalk-pkg --crosswalk=[crosswalk-version] --platform=android --android=shared --targets=[arch] /path/to/testapps/demo
  2. Install the built "demo" to 64-bit device(64-bit crosswalk runtime library installed following the table), and launch it.
  3. Uninstall the installed app

Expected Output:

  1. The apk named like "demo" is built successfully.
  2. The app named like "Demo V0.1" is installed and launched successfully.
  3. The app is uninstalled successfully.
